    About this blog

    Nova Blog is a hand-crafted, dependency-free demo. Content is rendered from markdown stored locally. All code is original and free (Unlicense).

    Features

    • Search & tag filters
    • Markdown rendering (headings, lists, links, code)
    • Hash router (#/post/<slug>)
    • Reading progress bar & reading time
    • Light/Dark theme with persistence
